# WasteFlow
A Web-Based Waste Collection Management System for Brgy. Ampayon, Butuan City

**Project Description**

WasteFlow is a web-based system designed to help organize and improve the waste collection process in Brgy. Ampayon, Butuan City. The system allows users and administrators to manage waste collection schedules, monitor reports, and improve communication between residents and waste collectors.

**Features**

User and Admin Log in
Waste Collection Scheduling
DashboardMonitorin
WasteCollectionReports
Notification System
Educational Content

HTML
CSS
JavaScript
PHP
MySQL

**Purpose of the System**
